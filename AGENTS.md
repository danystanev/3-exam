# Project Rules

Apply these rules to all work in this repository.

- Use HTML, CSS, JavaScript, and Bootstrap 5 to develop the web application.
- Reuse existing Bootstrap 5 components and the framework's JavaScript features instead of rebuilding common UI behavior from scratch.
- Use Node.js, npm, and Vite for the development environment.
- Keep the app modular. Split components and pages into separate files instead of putting large amounts of code in a single file.
- Preserve the existing Vite SPA structure and routing approach unless a change is explicitly requested.
